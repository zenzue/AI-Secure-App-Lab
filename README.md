# AI Secure App Lab

AI Secure App Lab is a hands-on developer-focused security lab for learning common vulnerabilities in LLM-powered applications.

The lab demonstrates insecure and secure implementations of AI application patterns including prompt handling, RAG, tool calling, memory, output rendering, and authorization.

This project is designed for educational use in secure AI development training. It does not use real secrets, real customer data, or production AI systems.

## Topics Covered

- Prompt Injection
- System Prompt Leakage
- Sensitive Information Disclosure
- RAG Indirect Prompt Injection
- Insecure Output Handling
- Excessive Agency
- Memory Poisoning
- Data and Model Poisoning
- Supply Chain Risks
- Unbounded Consumption

## Modes

- Vulnerable Mode
- Patched Mode

## Target Users

- Software Engineers
- AI Product Developers
- DevSecOps Engineers
- Security Analysts
